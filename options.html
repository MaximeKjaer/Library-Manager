  1. Select libraries



    Here you can select the libraries that you want to use. The app wil check online if a new version exists, download and show the library you have selected. Unselected libraries will be ignored.

  2. Refresh rate



    You can choose how often you want to check for new versions. I would suggest to check every hour.

  3. Badge notification



    Select badge notifications to see how many libraries have been updated.

  4. Desktop notifications



    You can choose to be notified on your desktop whenever the app detects a new version of one of the libraries you have chosen.

  5. FileSystem



    Click here to see the files stored in the FileSystem API.

  6. That's it
